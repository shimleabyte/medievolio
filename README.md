# Medievolio

A medieval-themed personal portfolio built as a second portfolio page to complement my first portfolio experience.

## Overview

Bookfolio presents my skills, projects, and contact details as chapters inside a fantasy-inspired codex. The idea is to keep the presentation creative and immersive while still staying professional and easy to navigate.

This project is designed to feel different from a standard portfolio site:
- the visual style follows a medieval/book aesthetic,
- the layout is narrative and thematic,
- and the overall experience is meant to reflect personality as well as technical ability.

## Why this project exists

My first portfolio page is more technical and direct. Bookfolio is intended to be a second, more expressive version — a creative companion that shows another side of my identity as a developer.

## Features

- Medieval-inspired visual theme
- Chapter-based sections for skills, projects, and contact
- Interactive reveal/gate animation
- Responsive layout for desktop and mobile
- Clean, minimal portfolio structure

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Project Structure

- `src/app/page.tsx` — main portfolio content
- `src/components/TomeGate.tsx` — interactive reveal component
- `src/app/layout.tsx` — root layout and metadata
- `src/app/globals.css` — global styles

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Then open http://localhost:3000 in your browser.

## Deployment

You can deploy this project using your preferred platform for Next.js applications, such as Vercel or any other hosting provider that supports Next.js.
